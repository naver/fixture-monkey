м•Ңм•„л‘җл©ҙ мўӢмқҖ мҡ©м–ҙ #

  • мғҳн”Ңл§Ғ(sampling): н…ҢмҠӨнҠёмҡ© лҚ°мқҙн„°лҘј мӢӨм ңлЎң л§Ңл“ңлҠ” кІғмқ„ мқҳлҜён•©лӢҲлӢӨ. sample() л©”м„ңл“ңлҘј нҳём¶ңн•  л•Ңл§ҲлӢӨ мғҲлЎңмҡҙ н…ҢмҠӨнҠё лҚ°мқҙн„°к°Җ мғқм„ұлҗ©лӢҲлӢӨ.
  • л№ҢлҚ”(builder): к°қмІҙлҘј лӢЁкі„м ҒмңјлЎң л§Ңл“Ө мҲҳ мһҲкІҢ лҸ„мҷҖмЈјлҠ” лҸ„кө¬мһ…лӢҲлӢӨ. Fixture Monkeyм—җм„ңлҠ” giveMeBuilder()лЎң л№ҢлҚ”лҘј мғқм„ұн•©лӢҲлӢӨ.
  • Path Expression: к°қмІҙмқҳ м–ҙл–Ө мҶҚм„ұмқ„ ліҖкІҪн• м§Җ м§Җм •н•ҳлҠ” л°©лІ•мһ…лӢҲлӢӨ. мҳҲлҘј л“Өм–ҙ “age"лҠ” лӮҳмқҙ мҶҚм„ұмқ„, “items[0]“мқҖ лҰ¬мҠӨнҠёмқҳ мІ« лІҲм§ё м•„мқҙн…ңмқ„, “address.city"лҠ” мЈјмҶҢ к°қмІҙ м•Ҳмқҳ лҸ„мӢң мҶҚм„ұмқ„ мқҳлҜён•©лӢҲлӢӨ.

  • API мҡ”м•Ҫ н‘ң

  • мһҗмЈј л¬»лҠ” м§Ҳл¬ё (FAQ)

API мҡ”м•Ҫ н‘ң #

кё°ліё API (мІҳмқҢ мӮ¬мҡ©н•ҳмӢңлҠ” л¶„л“Өмқ„ мң„н•ң н•„мҲҳ API) #

APIм„ӨлӘ…мҳҲмӢң мғҒнҷ©
set()мӣҗн•ҳлҠ” к°’ м§Ғм ‘ м§Җм •н•ҳкё°нҡҢмӣҗмқҳ лӮҳмқҙлҘј 20мӮҙлЎң м§Җм •
size()лҰ¬мҠӨнҠё нҒ¬кё° м§Җм •н•ҳкё°мһҘл°”кө¬лӢҲм—җ мғҒн’Ҳ 3к°ң лӢҙкё°
setNull()null к°’ м§Җм •н•ҳкё°нғҲнҮҙн•ң нҡҢмӣҗмқҳ мқҙл©”мқјмқ„ nullлЎң м„Өм •

нҷңмҡ© API (кё°ліё кё°лҠҘм—җ мқөмҲҷн•ҙм§„ нӣ„ мӮ¬мҡ©н•ҳм„ёмҡ”) #

APIм„ӨлӘ…мҳҲмӢң мғҒнҷ©
setInner()мһ¬мӮ¬мҡ© к°ҖлҠҘн•ң м„Өм • л§Ңл“Өкё°м—¬лҹ¬ н…ҢмҠӨнҠём—җм„ң к°ҷмқҖ нҳ•нғңмқҳ нҡҢмӣҗм •ліҙ мӮ¬мҡ©
setLazy()лҸҷм ҒмңјлЎң к°’ мғқм„ұн•ҳкё°мҲңм°Ём Ғмқё мЈјл¬ёлІҲнҳё мғқм„ұ
setPostCondition()мЎ°кұҙм—җ л§һлҠ” к°’ л§Ңл“Өкё°м„ұмқёл§Ң к°Җмһ… к°ҖлҠҘн•ң м„ңл№„мҠӨ н…ҢмҠӨнҠё
fixed()н•ӯмғҒ к°ҷмқҖ к°’ мғқм„ұн•ҳкё°н…ҢмҠӨнҠёл§ҲлӢӨ лҸҷмқјн•ң н…ҢмҠӨнҠё лҚ°мқҙн„° мӮ¬мҡ©
limitмқјл¶Җл§Ң к°’ м„Өм •н•ҳкё°мһҘл°”кө¬лӢҲмқҳ мқјл¶Җ мғҒн’Ҳл§Ң н• мқё м Ғмҡ©

кі кёү API (ліөмһЎн•ң н…ҢмҠӨнҠё мғҒнҷ©м—җм„ң мӮ¬мҡ©н•ҳм„ёмҡ”) #

APIм„ӨлӘ…мҳҲмӢң мғҒнҷ©
thenApply()м—°кҙҖлҗң к°’ м„Өм •н•ҳкё°мЈјл¬ё мҙқм•Ўмқ„ мЈјл¬ё мғҒн’Ҳ к°ҖкІ©мқҳ н•©мңјлЎң м„Өм •
customizeProperty()мҶҚм„ұ мғқм„ұ л°©мӢқ м„ёл°Җн•ҳкІҢ мЎ°м •н•ҳкё°мҶҚм„ұ мғқм„ұ л°©мӢқмқ„ м„ёл°Җн•ҳкІҢ мЎ°м •

кё°ліё API мӮ¬мҡ©н•ҳкё° #

set() #

set() л©”м„ңл“ңлҠ” к°қмІҙмқҳ нҠ№м • мҶҚм„ұм—җ мӣҗн•ҳлҠ” к°’мқ„ м„Өм •н•  л•Ң мӮ¬мҡ©н•©лӢҲлӢӨ. к°ҖмһҘ кё°ліём Ғмқҙкі  л§Һмқҙ мӮ¬мҡ©лҗҳлҠ” APIмһ…лӢҲлӢӨ.

кё°ліё мӮ¬мҡ©лІ• #
// нҡҢмӣҗ лҚ°мқҙн„° мғқм„ұ мҳҲм ң
Member member = fixtureMonkey.giveMeBuilder(Member.class)
    .set("name", "нҷҚкёёлҸҷ")        // мқҙлҰ„ м„Өм •
    .set("age", 25)             // лӮҳмқҙ м„Өм •
    .set("email", "hong@test.com") // мқҙл©”мқј м„Өм •
    .sample();

// мЈјл¬ё лҚ°мқҙн„° мғқм„ұ мҳҲм ң
Order order = fixtureMonkey.giveMeBuilder(Order.class)
    .set("orderId", "ORDER-001")           // мЈјл¬ёлІҲнҳё м„Өм •
    .set("totalAmount", BigDecimal.valueOf(15000)) // мЈјл¬ёкёҲм•Ў м„Өм •
    .sample();
// нҡҢмӣҗ лҚ°мқҙн„° мғқм„ұ мҳҲм ң
val member = fixtureMonkey.giveMeBuilder<Member>()
    .setExp(Member::name, "нҷҚкёёлҸҷ")        // мқҙлҰ„ м„Өм •
    .setExp(Member::age, 25)             // лӮҳмқҙ м„Өм •
    .setExp(Member::email, "hong@test.com") // мқҙл©”мқј м„Өм •
    .sample()

// мЈјл¬ё лҚ°мқҙн„° мғқм„ұ мҳҲм ң

Values.just() #

Values.just()лЎң лһҳн•‘лҗң к°қмІҙлҘј мӮ¬мҡ©н•ҳл©ҙ лӮҙл¶Җм ҒмңјлЎң к°қмІҙлҘј л¶„н•ҙн•ҳм§Җ м•Ҡкі  мӮ¬мҡ©мһҗк°Җ м ңкіөн•ң к°’мқ„ к·ёлҢҖлЎң мӮ¬мҡ©н•©лӢҲлӢӨ.

мЈјмқҳ: Just лЎң м„Өм •н•ң нӣ„м—җлҠ” н•ҳмң„ мҶҚм„ұмқ„ ліҖкІҪн•  мҲҳ м—ҶмҠөлӢҲлӢӨ.

Product product = fixture.giveMeBuilder(MyClass.class)
	  	.set("options", Values.just(List.of("red", "medium", "adult"))
	  	.set("options[0]", "blue")
        .sample();

мҳҲлҘј л“Өм–ҙ, MyClassмқҳ options[0] к°’мқҖ “blue” к°Җ м•„лӢҢ Values.just()лЎң м„Өм •лҗң лҰ¬мҠӨнҠёлЎң мң м§Җлҗ©лӢҲлӢӨ.

size(), minSize(), maxSize() #

size() л©”м„ңл“ңлҠ” лҰ¬мҠӨнҠёлӮҳ л°°м—ҙк°ҷмқҖ м»¬л үм…ҳмқҳ нҒ¬кё°лҘј м§Җм •н•  л•Ң мӮ¬мҡ©н•©лӢҲлӢӨ. м •нҷ•н•ң нҒ¬кё°лҘј м„Өм •н•ҳкұ°лӮҳ, мөңмҶҢ/мөңлҢҖ нҒ¬кё°лҘј м§Җм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

кё°ліё мӮ¬мҡ©лІ• #

Cart cart = fixtureMonkey.giveMeBuilder(Cart.class)
    .size("items", 3)  // мһҘл°”кө¬лӢҲм—җ 3к°ң мғҒн’Ҳ
    .sample();

// 2~4к°ң мӮ¬мқҙмқҳ лҰ¬л·°к°Җ мһҲлҠ” мғҒн’Ҳ л§Ңл“Өкё°
Product product = fixtureMonkey.giveMeBuilder(Product.class)
    .size("reviews", 2, 4)  // мөңмҶҢ 2к°ң, мөңлҢҖ 4к°ң лҰ¬л·°
    .sample();
// мһҘл°”кө¬лӢҲм—җ мғҒн’Ҳ 3к°ң лӢҙкё°
val cart = fixtureMonkey.giveMeBuilder<Cart>()
    .sizeExp(Cart::items, 3)  // мһҘл°”кө¬лӢҲм—җ 3к°ң мғҒн’Ҳ
    .sample()

// 2~4к°ң мӮ¬мқҙмқҳ лҰ¬л·°к°Җ мһҲлҠ” мғҒн’Ҳ л§Ңл“Өкё°
val product = fixtureMonkey.giveMeBuilder<Product>()

thenApply() #

thenApply()лҠ” мқҙлҜё мғқм„ұлҗң к°қмІҙмқҳ к°’мқ„ кё°л°ҳмңјлЎң лӢӨлҘё к°’мқ„ м„Өм •н•ҙм•ј н•  л•Ң мӮ¬мҡ©н•©лӢҲлӢӨ. мҳҲлҘј л“Өм–ҙ, мЈјл¬ёмқҳ мҙқм•Ўмқ„ мЈјл¬ё мғҒн’Ҳл“Өмқҳ к°ҖкІ© н•©кі„лЎң м„Өм •н•  л•Ң мң мҡ©н•©лӢҲлӢӨ.

кё°ліё мӮ¬мҡ©лІ• #
// мЈјл¬ё мғҒн’Ҳ к°ҖкІ©мқҳ н•©кі„лЎң мҙқм•Ў м„Өм •
Order order = fixtureMonkey.giveMeBuilder(Order.class)
    .size("items", 3)  // 3к°ң мғҒн’Ҳ
    .thenApply((tempOrder, orderBuilder) -> {
        // мҙқм•Ў кі„мӮ°
        BigDecimal total = tempOrder.getItems().stream()
            .map(item -> item.getPrice())
            .reduce(BigDecimal.ZERO, BigDecimal::add);
        // кі„мӮ°лҗң мҙқм•Ў м„Өм •
        orderBuilder.set("totalAmount", total);
    })
    .sample();
// мЈјл¬ё мғҒн’Ҳ к°ҖкІ©мқҳ н•©кі„лЎң мҙқм•Ў м„Өм •
val order = fixtureMonkey.giveMeBuilder<Order>()
    .sizeExp(Order::items, 3)  // 3к°ң мғҒн’Ҳ
    .thenApply { tempOrder, orderBuilder ->
        // мҙқм•Ў кі„мӮ°

            .map { it.price }
            .fold(BigDecimal.ZERO, BigDecimal::add)
        // кі„мӮ°лҗң мҙқм•Ў м„Өм •
        orderBuilder.set("totalAmount", total)
    }
    .sample()

customizeProperty() #

customizeProperty()лҠ” Fixture Monkeyк°Җ нҠ№м • мҶҚм„ұм—җ лҢҖн•ҙ к°’мқ„ мғқм„ұн•ҳлҠ” л°©мӢқмқ„ м„ёл°Җн•ҳкІҢ мЎ°м •н•  л•Ң мӮ¬мҡ©н•©лӢҲлӢӨ. мқҙлҠ” set()ліҙлӢӨ лҚ” кі кёү кё°лҠҘмңјлЎң, ліҖнҷҳкіј н•„н„°л§Ғмқ„ нҶөн•ҙ лҚ” л§ҺмқҖ м ңм–ҙк¶Ңмқ„ м ңкіөн•©лӢҲлӢӨ.

м–ём ң customizeProperty()к°Җ н•„мҡ”н•ңк°Җмҡ”? #

лӢӨмқҢкіј к°ҷмқҖ мғҒнҷ©м—җм„ң customizeProperty()к°Җ мң мҡ©н•©лӢҲлӢӨ:

  • мғқм„ұлҗң к°’мқ„ ліҖнҷҳн•ҳкі  мӢ¶мқ„ л•Ң: “лӘЁл“  мқҙлҰ„ м•һм—җ ‘лӢҳ’ л¶ҷмқҙкё°”
  • мЎ°кұҙл¶Җ н•„н„°л§Ғмқҙ н•„мҡ”н•  л•Ң: “м–‘мҲҳл§Ң н—Ҳмҡ©н•ҳкё°”
  • м»¬л үм…ҳм—җм„ң мң лӢҲнҒ¬н•ң к°’мқ„ мӣҗн•  л•Ң: “лҰ¬мҠӨнҠём—җ мӨ‘ліө н•ӯлӘ© м—Ҷм• кё°”
вҡ пёҸ
customizePropertyлҠ” TypedPropertySelectorк°Җ н•„мҡ”н•©лӢҲлӢӨ. кё°ліё APIмқё set(), size() л“ұм—җ мқөмҲҷн•ҙм§„ нӣ„ мӮ¬мҡ©н•ҳм„ёмҡ”.

мЈјмқҳмӮ¬н•ӯ #

  1. кё°ліё APIлҘј лЁјм Җ мқөнһҲм„ёмҡ”: customizeProperty()лҘј мӮ¬мҡ©н•ҳкё° м „м—җ set(), size(), setNull() л“ұмқ„ лЁјм Җ мқҙн•ҙн•ҳм„ёмҡ”

  2. н•„мҡ”н•ң нҒҙлһҳмҠӨлҘј import н•ҳм„ёмҡ”:

    // Javaмқҳ кІҪмҡ°
import static com.navercorp.fixturemonkey.api.experimental.JavaGetterMethodPropertySelector.javaGetter;

// мӢӨн—ҳм Ғ кё°лҠҘмқҳ кІҪмҡ°
import static com.navercorp.fixturemonkey.api.experimental.TypedExpressionGenerator.typedRoot;

  3. мҲңм„ңк°Җ мӨ‘мҡ”н•©лӢҲлӢӨ: set()мқҖ customizeProperty()лҘј л¬ҙмӢңн•©лӢҲлӢӨ

    // мҳҲмғҒлҢҖлЎң лҸҷмһ‘н•ҳм§Җ м•ҠмқҢ
.customizeProperty(javaGetter(Member::getName), arb -> arb.map(name -> "лӢҳ " + name))
.set("name", "нҷҚкёёлҸҷ")  // мң„мқҳ м»ӨмҠӨн„°л§Ҳмқҙм§•мқ„ л¬ҙмӢңн•Ё

  4. н•„н„° мЎ°кұҙмқ„ м Ғм ҲнһҲ м„Өм •н•ҳм„ёмҡ”: л„Ҳл¬ҙ к№ҢлӢӨлЎңмҡҙ мЎ°кұҙмқҖ мғқм„ұ мӢӨнҢЁлҘј мқјмңјнӮ¬ мҲҳ мһҲмҠөлӢҲлӢӨ

    // л„Ҳл¬ҙ к№ҢлӢӨлЎңмӣҖ - мӢӨнҢЁн•  мҲҳ мһҲмқҢ
.customizeProperty(javaGetter(Member::getAge), arb -> arb.filter(age -> age == 25))

// лҚ” лӮҳмқҢ - мң м—°н•ң лІ”мң„
.customizeProperty(javaGetter(Member::getAge), arb -> arb.filter(age -> age >= 20 && age <= 30))

  5. ліөмһЎн•ң ліҖнҷҳм—җл§Ң мӮ¬мҡ©н•ҳм„ёмҡ”: лӢЁмҲңнһҲ нҠ№м • к°’мқҙ н•„мҡ”н•ҳлӢӨл©ҙ set()мқ„ мӮ¬мҡ©н•ҳм„ёмҡ”

мһҗмЈј л¬»лҠ” м§Ҳл¬ё (FAQ) #

Q: м–ҙл–Ө APIл¶Җн„° л°°мӣҢм•ј н•ҳлӮҳмҡ”? #

мІҳмқҢм—җлҠ” лӢӨмқҢ мҲңм„ңлЎң л°°мҡ°мӢңлҠ” кІғмқ„ м¶”мІңн•©лӢҲлӢӨ:

  1. set() - к°ҖмһҘ кё°ліём Ғмқҙкі  л§Һмқҙ мӮ¬мҡ©лҗҳлҠ” APIмһ…лӢҲлӢӨ.
  2. size() - лҰ¬мҠӨнҠёлӮҳ л°°м—ҙмқ„ лӢӨлЈ° л•Ң н•„мҡ”н•©лӢҲлӢӨ.
  3. setNull(), setNotNull() - null к°’мқ„ лӢӨлЈ° л•Ң мӮ¬мҡ©н•©лӢҲлӢӨ.

мқҙнӣ„ н…ҢмҠӨнҠё мһ‘м„ұм—җ мқөмҲҷн•ҙм§Җл©ҙ лӢӨлҘё APIл“Өмқ„ н•ҳлӮҳм”© л°°мӣҢк°ҖмӢңл©ҙ лҗ©лӢҲлӢӨ.

Q: н…ҢмҠӨнҠёл§ҲлӢӨ к°ҷмқҖ лҚ°мқҙн„°к°Җ н•„мҡ”н•ҳл©ҙ м–ҙл–»кІҢ н•ҳлӮҳмҡ”? #

fixed()лҘј мӮ¬мҡ©н•ҳл©ҙ лҗ©лӢҲлӢӨ. мҳҲлҘј л“Өм–ҙ:

// н…ҢмҠӨнҠёл§ҲлӢӨ лҸҷмқјн•ң нҡҢмӣҗ м •ліҙ мӮ¬мҡ©
ArbitraryBuilder<Member> memberBuilder = fixtureMonkey.giveMeBuilder(Member.class)
    .set("name", "нҷҚкёёлҸҷ")
    .set("age", 30)
    .fixed();  // н•ӯмғҒ лҸҷмқјн•ң лҚ°мқҙн„° мғқм„ұ

Member member1 = memberBuilder.sample(); // н•ӯмғҒ к°ҷмқҖ лҚ°мқҙн„°
Member member2 = memberBuilder.sample(); // member1кіј лҸҷмқј

Q: мӢӨмҲҳлЎң мһҳлӘ»лҗң к°’мқҙ мғқм„ұлҗҳлҠ” кІғмқ„ л°©м§Җн•ҳл Өл©ҙ м–ҙл–»кІҢ н•ҳлӮҳмҡ”? #

setPostCondition()мқ„ мӮ¬мҡ©н•ҳм—¬ к°’мқҳ лІ”мң„лӮҳ мЎ°кұҙмқ„ м§Җм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ:

// лӮҳмқҙлҠ” л°ҳл“ңмӢң 1-100 мӮ¬мқҙ
Member member = fixtureMonkey.giveMeBuilder(Member.class)
    .setPostCondition("age", Integer.class, age -> age >= 1 && age <= 100)
    .sample();

Q: set()мҷҖ customizeProperty()мқҳ м°Ёмқҙм җмқҖ л¬ҙм—Үмқёк°Җмҡ”? #

  • set()мқҖ нҠ№м • к°’мқ„ м§Ғм ‘ н• лӢ№н•©лӢҲлӢӨ
  • customizeProperty()лҠ” мҶҚм„ұ к°’мқҙ мғқм„ұлҗҳлҠ” л°©мӢқмқ„ мҲҳм •н•ҳм—¬ н•„н„°л§Ғ, ліҖнҷҳ, мЎ°кұҙл¶Җ лЎңм§Ғмқҙ к°ҖлҠҘн•©лӢҲлӢӨ

м •нҷ•н•ң к°’мқ„ м•Ңкі  мһҲлӢӨл©ҙ set()мқ„, мғқм„ұлҗң к°’м—җ ліҖнҷҳмқҙлӮҳ н•„н„°лҘј м Ғмҡ©н•ҙм•ј н•ңлӢӨл©ҙ customizeProperty()лҘј мӮ¬мҡ©н•ҳм„ёмҡ”:

// м§Ғм ‘ н• лӢ№ - set() мӮ¬мҡ©
Member member = fixtureMonkey.giveMeBuilder(Member.class)
    .set("name", "нҷҚкёёлҸҷ")  // м •нҷ•нһҲ "нҷҚкёёлҸҷ"мңјлЎң м„Өм •
    .sample();

// ліҖнҷҳ/н•„н„°л§Ғ - customizeProperty() мӮ¬мҡ©  
Member adultMember = fixtureMonkey.giveMeBuilder(Member.class)
    .customizeProperty(javaGetter(Member::getAge), arb -> 
        arb.filter(age -> age >= 18)  // 18м„ё мқҙмғҒл§Ң н—Ҳмҡ©
           .map(age -> age + 10))     // лӘЁл“  лӮҳмқҙм—җ 10мқ„ лҚ”н•Ё
