복잡한 객체를 자동으로 만들어 테스트를 편하게 만들어주는 라이브러리

{{< blocks/link-down color="info" >}} {{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Fixture Monkey는 테스트 객체를 편하게 자동으로 생성해주는 자바 라이브러리 입니다.

JVM 생태계에서 더 편하게 테스트를 작성하고 더 편하게 테스트를 읽을 수 있는 환경을 만들고자 합니다. {{% /blocks/lead %}} {{% blocks/section color="dark" type="row" %}} {{% blocks/feature icon="fas fa-recycle" title="재사용성" %}} 복잡한 스펙의 테스트 객체를 재사용할 수 있습니다. 테스트마다 복잡한 객체를 매번 정의할 필요 없습니다. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-redo" title="반복성" %}} 테스트마다 새로운 객체를 자동으로 생성해서 테스트할 수 있습니다. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-wrench" title="간단함" %}} 빌더 패턴으로 복잡한 테스트 객체를 정의할 수 있습니다. 테스트 객체 내 복잡한 연관관계도 쉽게 처리할 수 있습니다. {{% /blocks/feature %}} {{% /blocks/section %}}
