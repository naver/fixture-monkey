# 릴리스 노트

메이저 버전별로 정리된 Fixture Monkey 릴리스 노트입니다. 각 페이지에는 유형별로 분류된 상세 변경 사항이 포함되어 있습니다.

## 최신: v1.1.17

- `JavaNodeTreeAdapterPlugin` / `KotlinNodeTreeAdapterPlugin` — 제로 오버헤드와 내장 트레이싱을 갖춘 대체 객체 생성 엔진
- `object-farm-api` 모듈 — JvmNodeTree, PathExpression, 타입 시스템 인프라

전체 변경 로그는 [v1.1.x](./v1.1.x)를 참조하세요.
