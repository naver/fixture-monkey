# 릴리스 노트

메이저 버전별로 정리된 Fixture Monkey 릴리스 노트입니다. 각 페이지에는 유형별로 분류된 상세 변경 사항이 포함되어 있습니다.

## 최신: v1.1.16

- Jackson3 지원 모듈
- 추가 `CombinableArbitrary` 타입 (`Float`, `Double`, `BigInteger`, `BigDecimal`, `Character`, `Short`, `Long`)
- JSpecify 어노테이션 지원
- Gradle 버전 카탈로그로 마이그레이션

전체 변경 로그는 [v1.1.x](./v1.1.x)를 참조하세요.
