InnerSpecмқҖ м Ғмҡ©н•ҳл ӨлҠ” м»ӨмҠӨн„°л§Ҳмқҙм§•м—җ лҢҖн•ң нғҖмһ… лҸ…лҰҪм Ғмқё лӘ…м„ёмһ…лӢҲлӢӨ. ArbitraryBuilder лӮҙмқҳ setInner() л©”м„ңл“ңлҘј мӮ¬мҡ©н•ҳл©ҙ InnerSpec мқёмҠӨн„ҙмҠӨм—җ м •мқҳлҗң лӘ…м„ёлҘј л№ҢлҚ”м—җ м Ғмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

InnerSpec м—җлҠ” м»ӨмҠӨн„°л§Ҳмқҙм§• м„ёл¶Җ м •ліҙк°Җ м ҖмһҘлҗҳл©° м—¬лҹ¬ ArbitraryBuilderм—җм„ң мһ¬мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. ArbitraryBuilderм—җм„ң н”ҪмҠӨмІҳ лӘҪнӮӨ н‘ңнҳ„мӢқмқ„ мӮ¬мҡ©н•ҳлҠ” л°©мӢқкіј лӢ¬лҰ¬ InnerSpecмқҖ мӨ‘мІ©лҗң кө¬мЎ°лҘј мӮ¬мҡ©н•ҙ н”„лЎңнҚјнӢ°м—җ м ‘к·јмқҙ к°ҖлҠҘн•©лӢҲлӢӨ.

InnerSpec мқҳ лҳҗ лӢӨлҘё мһҘм җмқҖ мқјл°ҳм Ғмқё н‘ңнҳ„мӢқкіј лӢ¬лҰ¬ л§ө н”„лЎңнҚјнӢ°лҘј м»ӨмҠӨн„°л§Ҳмқҙм§•н•  мҲҳ мһҲлӢӨлҠ” м җмһ…лӢҲлӢӨ.

рҹ’Ў
Kotlin EXP лҠ” InnerSpecм—җм„ң м§Җмӣҗн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ. InnerSpecмқҖ нғҖмһ… лҸ…лҰҪм ҒмңјлЎң м„Өкі„лҗҳм—Ҳкё° л•Ңл¬ём—җ, н”„лЎңнҚјнӢ° мқҙлҰ„мқ„ нҶөн•ҙ н”„лЎңнҚјнӢ°лҘј м ‘к·јн•ҙм•ј н•©лӢҲлӢӨ.

ArbitraryBuilder м—җ InnerSpec м Ғмҡ©н•ҳкё° #

л№ҢлҚ”м—җ лҜёлҰ¬ м •мқҳлҗң InnerSpec мқ„ м Ғмҡ©н•ҳл Өл©ҙ лӢӨмқҢкіј к°ҷмқҙ setInner() л©”м„ңл“ңлҘј мӮ¬мҡ©н•ҳм„ёмҡ”.

InnerSpec innerSpec = new InnerSpec().property("id", 1000);

fixtureMonkey.giveMeBuilder(Product.class)
    .setInner(innerSpec);
val innerSpec = InnerSpec().property("id", 1000)

fixtureMonkey.giveMeBuilder<Product>()
    .setInner(innerSpec)

н”„лЎңнҚјнӢ° м»ӨмҠӨн„°л§Ҳмқҙм§•н•ҳкё° #

property() #

ArbitraryBuilder мқҳ set() л©”м„ңл“ңмҷҖ мң мӮ¬н•ҳкІҢ, н”„лЎңнҚјнӢ° мқҙлҰ„кіј мӣҗн•ҳлҠ” к°’мқ„ м§Җм •н•ҳм—¬ н”„лЎңнҚјнӢ°лҘј м»ӨмҠӨн„°л§Ҳмқҙм§•н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

рҹҡЁ
мҡ”мҶҢ(`[]`) лҳҗлҠ” мӨ‘мІ© н•„л“ң(`.`)лҘј м°ёмЎ°н•ҳлҠ” Fixture Monkey н‘ңнҳ„мӢқмқҖ н”„лЎңнҚјнӢ° мқҙлҰ„мңјлЎң мӮ¬мҡ©н•  мҲҳ м—ҶмҠөлӢҲлӢӨ. н”„лЎңнҚјнӢ° мқҙлҰ„ мһҗмІҙл§Ң мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.
InnerSpec innerSpec = new InnerSpec()
    .property("id", 1000);
val innerSpec = InnerSpec()
    .property("id", 1000)

size(), minSize(), maxSize() #

size(), minSize(), к·ёлҰ¬кі  maxSize() лҠ” н”„лЎңнҚјнӢ°мқҳ нҒ¬кё°лҘј м§Җм •н•ҳлҠ” лҚ° мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

м•һм„ң м–ёкёүн–Ҳл“Ҝмқҙ, InnerSpec мқҖ мӨ‘мІ©лҗң л°©мӢқмңјлЎң лӘ…м„ёмқ„ м •мқҳн•©лӢҲлӢӨ. property() лҘј мӮ¬мҡ©н•ҳм—¬ м»Ён…Ңмқҙл„Ҳ н”„лЎңнҚјнӢ°лҘј лЁјм Җ м„ нғқн•ң лӢӨмқҢ, лӮҙл¶Җм—җ м •мқҳлҗң innerSpec м»ЁмҠҲлЁёлҘј мӮ¬мҡ©н•ҳм—¬ нҒ¬кё°лҘј м„Өм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

InnerSpec innerSpec = new InnerSpec()
    .property("options", options -> options.size(5)); // size:5

InnerSpec innerSpec = new InnerSpec()
    .property("options", options -> options.size(3, 5)); // minSize:3, maxSize:5

InnerSpec innerSpec = new InnerSpec()
    .property("options", options -> options.minSize(3)); // minSize:3

InnerSpec innerSpec = new InnerSpec()
    .property("options", options -> options.maxSize(5)); // maxSize:5

val innerSpec = InnerSpec()
    .property("options") { it.size(5) } // size:5

val innerSpec = InnerSpec()
    .property("options") { it.size(3, 5) } // minSize:3, maxSize:5

val innerSpec = InnerSpec()
    .property("options") { it.minSize(3) } // minSize:3

val innerSpec = InnerSpec()
    .property("options") { it.maxSize(5) } // maxSize:5

postCondition() #

postCondition() мқҖ н”„лЎңнҚјнӢ°к°Җ нҠ№м • мЎ°кұҙмқ„ л§ҢмЎұн•ҙм•ј н•ҳлҠ” кІҪмҡ° мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

рҹҡЁ
setPostCondition мқҳ мЎ°кұҙмқ„ л„Ҳл¬ҙ мўҒкІҢ м„Өм •н•ҳл©ҙ, мғқм„ұ л№„мҡ©мқҙ л§Өмҡ° лҶ’м•„м§Ҳ мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлҹ° кІҪмҡ° set мқ„ мӮ¬мҡ©н•ҙмЈјм„ёмҡ”.
InnerSpec innerSpec = new InnerSpec()
    .property("id", id -> id.postCondition(Long.class, it -> it > 0));
val innerSpec = InnerSpec()
    .property("id") { it.postCondition(Long::class.java) { it > 0 }}

inner() #

лҳҗн•ң inner() лҘј мӮ¬мҡ©н•ҳм—¬ лҜёлҰ¬ м •мқҳлҗң InnerSpec мқ„ мӮ¬мҡ©н•ҳм—¬ н”„лЎңнҚјнӢ°лҘј м»ӨмҠӨн„°л§Ҳмқҙм§•н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

InnerSpec innerSpec = new InnerSpec()
    .property("id", 1000L);

fixtureMonkey.giveMeBuilder(Product.class)
    .setInner(
        new InnerSpec()
            .property("nestedObject", nestedObject -> nestedObject.inner(innerSpec))
    );
val innerSpec = InnerSpec()
    .property("id", 1000L)

fixtureMonkey.giveMeBuilder<Product>()
    .setInner(
        InnerSpec()
            .property("nestedObject") { it.inner(innerSpec) }
    )

лҰ¬мҠӨнҠё м»ӨмҠӨн„°л§Ҳмқҙм§•н•ҳкё° #

listElement() #

лӘ©лЎқ лӮҙмқҳ к°ңлі„ мҡ”мҶҢлҠ” listElement()лҘј мӮ¬мҡ©н•ҳм—¬ м„ нғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлҠ” н”ҪмҠӨмІҳ лӘҪнӮӨ н‘ңнҳ„мӢқмқ„ мӮ¬мҡ©н•ҳм—¬ “[n]“мңјлЎң мҡ”мҶҢлҘј м°ёмЎ°н•ҳлҠ” кІғкіј лҸҷмқјн•©лӢҲлӢӨ.

InnerSpec innerSpec = new InnerSpec()
    .property("options", options -> options.listElement(0, "red"));
val innerSpec = InnerSpec()
    .property("options") { it.listElement(0, "red") }

allListElement() #

л§Ңм•Ҫ лӘ©лЎқмқҳ лӘЁл“  мҡ”мҶҢлҘј лҸҷмӢңм—җ м„Өм •н•ҳл Өл©ҙ allListElement()лҘј мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлҠ” н”ҪмҠӨмІҳ лӘҪнӮӨ н‘ңнҳ„мӢқмқ„ мӮ¬мҡ©н•ҳм—¬ “[*]“лЎң мҡ”мҶҢлҘј м°ёмЎ°н•ҳлҠ” кІғкіј лҸҷмқјн•©лӢҲлӢӨ.

л§ө м»ӨмҠӨн„°л§Ҳмқҙм§•н•ҳкё° #

InnerSpecмқҖ л§ө н”„лЎңнҚјнӢ° м—”нҠёлҰ¬лҘј м»ӨмҠӨн„°л§Ҳмқҙм§•н•ҳкё° мң„н•ҙ нҠ№лі„н•ң л©”мҶҢл“ңлҘј м ңкіөн•©лӢҲлӢӨ.

рҹҡЁ
л§ө н”„лЎңнҚјнӢ°мқҳ нҒ¬кё°лҘј лЁјм Җ м§Җм •н•ҳм§Җ м•Ҡкі  л§ө м—”нҠёлҰ¬лҘј м„Өм •н•ҳл©ҙ ліҖкІҪмқҙ мқјм–ҙлӮҳм§Җ м•Ҡмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ. к°’мқ„ м„Өм •н•ҳкё° м „м—җ л§ө н”„лЎңнҚјнӢ°к°Җ мқҳлҸ„н•ң нҒ¬кё°мқём§Җ нҷ•мқён•ҙмЈјм„ёмҡ”.

key(), value(), entry() #

key(), value(), entry() л©”мҶҢл“ңлҘј мӮ¬мҡ©н•ҳм—¬ л§ө н”„лЎңнҚјнӢ° м—”нҠёлҰ¬лҘј м»ӨмҠӨн„°л§Ҳмқҙм§•н•  мҲҳ мһҲмҠөлӢҲлӢӨ. key()лҘј мӮ¬мҡ©н•ҳл©ҙ л§ө м—”нҠёлҰ¬мқҳ нӮӨм—җ м§Җм •лҗң к°’мқ„ н• лӢ№н•ҳкі , м—”нҠёлҰ¬мқҳ к°’мқҖ л¬ҙмһ‘мң„лЎң м„Өм •лҗ©лӢҲлӢӨ. л§Ҳм°¬к°Җм§ҖлЎң, value()лҘј мӮ¬мҡ©н•ҳл©ҙ л§ө м—”нҠёлҰ¬мқҳ к°’м—җ м§Җм •лҗң к°’мқ„ н• лӢ№н•ҳкі , нӮӨлҠ” л¬ҙмһ‘мң„лЎң м„Өм •лҗ©лӢҲлӢӨ. нӮӨмҷҖ к°’мқ„ лҸҷмӢңм—җ м§Җм •н•ҳл Өл©ҙ entry()лҘј мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

InnerSpec innerSpec = new InnerSpec()

л©”м„ңл“ңлҘј мЎ°н•©н•ҳм—¬ InnerSpec лӮҙм—җм„ң л§өмқҳ нӮӨ, к°’ лҳҗлҠ” л‘ҳ лӢӨлҘј нҡЁкіјм ҒмңјлЎң м»ӨмҠӨн„°л§Ҳмқҙм§•н•  мҲҳ мһҲмҠөлӢҲлӢӨ.
